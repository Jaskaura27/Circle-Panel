# Circle-Panel

> Create, sort and recentre colored circles.

![](image1.png)
## Fileds
* Number of Circles:
  * Entre the number of circles you want to delete.
* Radius of Smallest Circle:
  * Entre minimum radius.
* Radius of Biggest Circle:
  * Entre maximum radius.
